# Personal Portfolio Website - React.js

A modern personal portfolio website built with React.js and React Router.

## Features

- Home page with personal information and social links
- About page with navigation
- Contact page
- Responsive design
- Modern React architecture

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
